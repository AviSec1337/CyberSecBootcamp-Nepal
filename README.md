# Cyber Security Week Course - Grade 9 & 10 Students
## 1-Hour Daily Sessions (Sunday to Friday)

---

## **Day 1 (Sunday): Introduction to Cybersecurity & Digital Safety**

### Topics to Cover (50 mins):
1. **What is Cybersecurity?** (10 mins)
   - CIA Triad: Confidentiality, Integrity, Availability
   - Why cybersecurity matters in daily life
   - Real-world examples students can relate to

2. **Types of Cyber Threats** (15 mins)
   - Malware (viruses, trojans, ransomware)
   - Phishing and social engineering
   - Password attacks
   - Show recent Nepal cyber attack examples

3. **Digital Safety Basics** (15 mins)
   - Strong passwords & password managers
   - Two-factor authentication (2FA)
   - Safe browsing habits
   - Social media privacy

4. **Nepal Cybersecurity Context** (10 mins)
   - Recent attacks on Nepali organizations
   - Common threats facing Nepali users
   - Government initiatives

### Interactive Quiz (10 mins):
**"Cyber Detective Challenge"**
- Spot the phishing email (show 3 emails, identify the fake)
- Password strength game (rate passwords as weak/medium/strong)
- True/False rapid fire about cyber threats
- Scenario: "Your friend sends you a link to download free game hacks. What do you do?"

---

## **Day 2 (Monday): Network Fundamentals**

### Topics to Cover (50 mins):
1. **How the Internet Works** (15 mins)
   - What is a network?
   - IP addresses and MAC addresses
   - DNS (Domain Name System)
   - Routers, switches, and firewalls

2. **Network Protocols** (15 mins)
   - HTTP vs HTTPS (show the lock icon!)
   - TCP/IP basics
   - Ports and services
   - Introduction to packets

3. **Network Security Basics** (10 mins)
   - Public vs Private networks
   - WiFi security (WPA2, WPA3)
   - VPNs - what and why?
   - Man-in-the-Middle attacks

4. **Hands-on Activity** (10 mins)
   - Check your own IP address
   - Use `ping` command demonstration
   - Show how to check if a website is secure

### Interactive Quiz (10 mins):
**"Network Navigator Game"**
- Match the protocol to its purpose
- IP address challenge: Identify public vs private IPs
- Port number matching game (80‚ÜíHTTP, 443‚ÜíHTTPS, etc.)
- Scenario quiz: "You're at a caf√©. Should you do online banking on their free WiFi?"
- WiFi security ranking exercise

---

## **Day 3 (Tuesday): Linux Fundamentals**

### Topics to Cover (50 mins):
1. **Introduction to Linux** (10 mins)
   - What is Linux and why it's important in cybersecurity
   - Linux distributions (Ubuntu, Kali Linux)
   - Difference between Windows and Linux

2. **Basic Linux Commands** (25 mins)
   - Navigation: `pwd`, `ls`, `cd`
   - File operations: `mkdir`, `touch`, `cp`, `mv`, `rm`
   - Viewing files: `cat`, `less`, `head`, `tail`
   - System info: `whoami`, `uname`, `df`, `free`
   - Permissions basics: `chmod`, `chown`

3. **Live Command Demonstrations** (15 mins)
   - Use online Linux terminal (JSLinux or similar)
   - Students follow along with basic commands
   - Create a folder structure exercise

### Interactive Quiz (10 mins):
**"Linux Command Quest"**
- Command matching game (match command to function)
- "Fix the command" - spot the errors in commands
- Scenario: "You need to create 3 folders and a text file. What commands do you use?"
- Permission puzzle: What does `rwxr-xr--` mean?
- Speed round: Name 5 Linux commands as fast as possible!

---

## **Day 4 (Wednesday): Offensive Security - Understanding Attacks**

### Topics to Cover (50 mins):
1. **What is Offensive Security?** (10 mins)
   - Ethical hacking vs malicious hacking
   - Penetration testing
   - Bug bounty programs

2. **Common Attack Types** (20 mins)
   - **Social Engineering**: Phishing, pretexting, baiting
   - **Malware Attacks**: How viruses spread
   - **Web Attacks**: SQL injection basics, XSS (simplified)
   - **Network Attacks**: DDoS, packet sniffing
   - **Password Attacks**: Brute force, dictionary attacks

3. **Attack Lifecycle** (10 mins)
   - Reconnaissance
   - Scanning
   - Exploitation
   - Post-exploitation
   - Covering tracks

4. **Nepal Case Studies** (10 mins)
   - NTC data breach incidents
   - Banking sector phishing attempts
   - Government website defacements
   - Student data leaks

### Interactive Quiz (10 mins):
**"Hacker's Mindset Challenge"**
- Attack type identification (describe scenario, name the attack)
- "Spot the vulnerability" in simple code snippets
- Role-play: You're a penetration tester - what steps would you take?
- Nepal-specific: Match the attack to the real incident
- Ethical dilemma questions: What would you do if you found a vulnerability?

---

## **Day 5 (Thursday): Defensive Security - Protection & Response**

### Topics to Cover (50 mins):
1. **What is Defensive Security?** (10 mins)
   - Blue team vs Red team
   - Security Operations Center (SOC)
   - Incident response

2. **Defense Mechanisms** (20 mins)
   - **Firewalls**: How they protect networks
   - **Antivirus & Anti-malware**: How they work
   - **Intrusion Detection Systems (IDS)**
   - **Encryption**: Protecting data at rest and in transit
   - **Backups**: The 3-2-1 rule
   - **Security Updates**: Why they matter

3. **Best Security Practices** (10 mins)
   - Principle of least privilege
   - Defense in depth
   - Security awareness training
   - Incident response steps

4. **Building a Security Mindset** (10 mins)
   - How to think defensively
   - Security for students: protecting your devices
   - What to do if you're attacked

### Interactive Quiz (10 mins):
**"Cyber Defender Battle"**
- Defense mechanism matching game
- "Build your defense" - layer security controls for a scenario
- Incident response ordering (arrange steps in correct order)
- True/False: Common security myths
- Team challenge: Design a security policy for your school's computer lab

---

## **Day 6 (Friday): Capstone Project & Career Exploration**

### Topics to Cover (40 mins):
1. **Review Key Concepts** (10 mins)
   - Quick recap of the week
   - Q&A session

2. **Mini Capture The Flag (CTF) Challenge** (20 mins)
   - Simple puzzles combining all concepts:
     - Decode a message (basic cryptography)
     - Find hidden information in a file
     - Identify vulnerabilities in scenarios
     - Answer trivia questions to unlock "flags"

3. **Cybersecurity Careers** (10 mins)
   - Penetration tester
   - Security analyst
   - Network security engineer
   - Cybersecurity consultant
   - Digital forensics expert
   - Path to become a cybersecurity professional in Nepal

### Final Interactive Quiz & Competition (20 mins):
**"Ultimate Cyber Champion Quiz Show"**
- Team-based competition
- Mix of questions from entire week
- Rapid-fire rounds
- Practical scenarios
- Bonus points for creative answers
- Prize for winning team (certificate/badge)

**Quiz Categories:**
- Network Ninjas (networking questions)
- Linux Legends (command-line challenges)
- Attack Awareness (offensive security)
- Defense Masters (defensive security)
- Nepal Cyber Facts (local context)

---

## **Additional Resources for Each Day:**

### Nepal Cybersecurity Facts to Include:
- Nepal Police Cyber Bureau cases
- Recent data breaches in Nepali organizations
- Common scams targeting Nepali users (Khalti/eSewa phishing, fake scholarship schemes)
- Government initiatives like Nepal Rastra Bank's security guidelines

### Engagement Strategies:
1. **Daily "Hack of the Day"**: Start each class with a 2-minute cool cyber fact
2. **Points System**: Award points for quiz participation, track leaderboard
3. **Real Device Demos**: Show attacks/defenses on actual devices when possible
4. **Student Examples**: Use apps they know (TikTok, Instagram, PUBG) in examples
5. **Create a Class Discord/WhatsApp**: Share resources and continue discussions

### Assessment:
- Daily quiz scores (10 points each)
- Final CTF challenge (30 points)
- Class participation (10 points)
- Certificate for students scoring 70%+
